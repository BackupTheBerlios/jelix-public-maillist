Bonjour,

Je débute avec Jelix, j'écris ma première application.

J'ai un module appelé «page» qui sert à afficher le texte des pages du
site.

Mon fichier page.dao.xml est le suivant:

<?xml version="1.0" encoding="UTF-8"?>
<dao xmlns="http://jelix.org/ns/dao/1.0">
  <datasources>
    <primarytable name="pages" primarykey="id" />
  </datasources>
  <record>
    <property
      name="id"
      fieldname="id"
      datatype="string"
      required="true"
      maxlength="255"
    />
    <property
      name="title"
      fieldname="title"
      datatype="string"
      required="true"
      maxlength="255"
    />
    <property
      name="text"
      fieldname="text"
      datatype="text"
      required="true"
    />
    <property
      name="date_creation"
      fieldname="date_creation"
      datatype="datetime"
      required="true"
      insertpattern="NOW()"
      updatepattern=""
    />
    <property
      name="date_modification"
      fieldname="date_modification"
      datatype="datetime"
      required="true"
      insertpattern="NOW()"
      updatepattern="NOW()"
    />
  </record>
  <factory>
  </factory>
</dao>

J'ai utilisé insertpattern et updatepattern pour que les dates de
création et de mise à jour soient renseignée automagicalement, mais je
me demande comment je vais bien pouvoir faire pour l'ID de mes pages, ce
serait bien s'il y avait quelque chose de similaire.

Je n'ai pas voulu utiliser un ID numérique avec auto_increment, mais
plutôt un ID sous forme de VARCHAR pour que mes URL soient mieux. Par
exemple, si j'ai une page dont le titre est «Ça c'est un joli titre!»,
son ID serait «ca_c_est_un_joli_titre» et comme ça, au lieu d'avoir des
URL du type http://www.example.com/myapp/www/?page=1, j'aurai plutôt
quelque chose qui ressemblerait à
http://www.example.com/myapp/www/?page=ca_c_est_un_joli_titre.

Mon problème c'est que je n'ai pas encore bien déterminé comment je
pourrais générer ces ID à partir du titre en étant sûr qu'ils sont
uniques (mais ça je trouverai toujours bien), mais surtout, je ne sais
pas où je dois mettre le code qui génère cet ID...

Quelqu'un aurait-il une idée?

Merci d'avance!
