Bonjour,

Afin de mieux suivre les développements du framework, chaque 
modification dans le dépot subversion sera publié dans la mailing list. 
(message du même style que le précedent : "checkin SVN-XXX (auteur)" )

Rassurez vous, le nombre de message par jour ne va pas exploser ( 0 à 5 
messages, selon les jours..).

Cela va permettre de mieux informer quand le trunk est instable, et de 
signaler les modifications importantes qui peuvent casser la 
compatibilité (en version alpha, tout est permis :-) ) etc, le tout en 
temps réèl, la page http://jelix.org/articles/changelog n'étant pas 
toujours à jour.

À ce propos, je vous invite à la lire, pour voir les modifications que 
l'on a apporté depuis la version 1.0alpha4.

Prochaine release d'une version "officielle" : ce n'est pas encore 
vraiment décidé. En tout cas, il y aura probablement une surprise 
courant/fin septembre :-)
