Bonjour,

Voici les changements qu'il y a eu cette semaine :

- nombreux bugs corrigés notament dans les daos, le driver db postgresql 
(merci à Giviz pour ses nombreux tests)
- d'autres bugs notament corrigés par dmp
- quelques modifications pour pouvoir utiliser les classes de jelix hors 
web, et donc notamment dans les scripts en ligne de commande.
- commande de script createdao, maintenant operationnelle
   jelix.php createdao monModule nomDao uneTable

   créer un fichier xml de dao mappé sur la table uneTable


Il n'y a pas eu donc énormément d'évolutions cette semaine. J'ai plutôt 
porté mes efforts sur la documentation sur www.jelix.org , avec un 
tutoriel "rapide" donnant un aperçu de la structure d'une appli, et le 
commencement d'un tutoriel assez complet.
Vous trouverez également une roadmap approximative...

Sortie le 16 en soirée d'une version 1.0pre-alpha.

a+
