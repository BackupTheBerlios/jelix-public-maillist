Hello

a new mailing list has been created : jelix-dev . With this mailing 
list, you receive all automatic mails :

- commits
- nightlies building
- results of automatic unit tests
- results of builds of PDFs and API reference

And it is dedicated also for technical discussion about development of 
jelix. So all members of the jelix team and active contributors are 
encouraged to subscribe to this mailing-list.

Jelix-public is now dedicated exclusively for discussions about the use 
of Jelix.
